# HR Analytics Dashboard

## Project Overview
Analysis of employee attrition data from a company
of 1,470 employees to identify key factors driving
employee turnover.

## Tools Used
- Microsoft Excel (data cleaning)
- Power BI (dashboard & visualization)
- DAX (measures & calculations)

## Key Insights
- R&D department has the highest attrition (133 employees)
- Laboratory Technicians leave the most
- Employees aged 28-35 are the highest risk group
- Males leave more than females (63% vs 37%)
- Overtime significantly increases attrition
